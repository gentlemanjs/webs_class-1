반복문

반복문은 코드를 원하는 횟수만큼 반복하여 실행 할 수 있게합니다.
"안녕"이라는 문구를 100회 출력하려면 출력문을 100번 작성해야하는 대신 반복문을 사용하면 한 줄의 출력문으로 100회 출력할 수 있습니다.
반복문의 종류에는 while문,for문,break문,continue문 등이 있습니다.

while문

while문은 조건식을 만족할 때까지 코드를 여러 회 반복하여 실행할 수 있습니다.

while문의 기본형

{
    var 변수 = 초기값;
    while(조건식) {
    증감식;
    }
}

for문

for문은 조건식을 만족할 때까지 특정 코드를 반복하여 실행합니다.

for문의 기본형

{
    for (초기값; 조건식; 증감식;) {
        자바스크립트 코드;
    }
}

break문

break문은 while문 or for문에서 break문을 실행하면 조건식과 상관없이 강제로 반복문을 합니다.

break문의 기본형

{
    for (초기값; 조건식; 증감식;) {
        break; //반복문을 강제로 종료합니다.
        자바스크립트 코드;
    }

    var 변수 = 초기값;
    while(조건식) {
        break;
        continue;
        증감식;
    }
}

continue문

continue문은 반복문에서만 사용할 수 있습니다.

while문에서 사용할 경우 continue문 다음에 오는 코드는 무시하고 바로 '조건식'으로 이동해 조건 검사를 합니다.
for문에서 사용할 경우 continue문 다음에 오는 코드는 무시하고 바로 '증감식'으로 이동해 증감 연산을 합니다.

continue문의 기본형

{
    for (초기값; 조건식; 증감식;) {
        continue; 
        자바스크립트
    }

    var 변수 = 초기값;
    while(조건식) {
        증감식;
        continue;
        자바스크립트 코드;
    }
}
