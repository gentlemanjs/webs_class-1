자바스크립트 선언문

선언문이란 '자바스크립트 코드를 작성'할 영역을 선언하는 것
스크립트 영역 : <script>라는 태크로 선언문이 시작된 곳부터 </script>라는 태그로 종료된 곳까지 

{
    <script>
      자바스크립트 코드;
    </script>
}

자바스크립트 주석처리

자바스크립트 선언문 안에서 코드가 아닌 설명글을 넣고 싶을 떄 주석 처리한다.(단축키 : ctrl + /)
한 줄 주석일 경우 : //(한줄 설명글)로 작성
여러 줄 주석일 경우 : /*(여러줄 설명글)*/로 작성 

{
    //한 줄 설명글인 경우
    /*
    설명글이 여러 줄인 경우
    이렇게 처리합니다
    */
}

코드 입력시 주의사항

자바스크립트를 작성할 때 몇 주의 사항

  • 자바스크립트 대소문자를 구분하여 작성합니다.
    • 날짜 객체 생성 : New date(); : (x)
날짜 객체 생성 : New Date(); : (O)
  • 코드 한 줄 작성 후에는 세미콜론(;)을 쓴다. 쓰지 않으면 한 줄에 2개의 코드를 작성할 경우 오류 발생
    • document.write("hi") document.write("bye") : (x)
document.write("hi"); document.write("bye"); : (O)
  • 코드를 작성할 때는 한 줄에 한 문장만 작성하는 것이 가독성을 위해 좋다.
    • BAD : document.write("hi"); document.write("bye"); 
GOOD :            
 document.write("hi"); 
 document.write("bye");
  • 문자형 데이터를 작성할 때: 큰따옴표("")와 작은따옴표('')의 겹침 오류 주의
    • 큰 따옴표 겹침 오류
document.write("겹침 "오류를 주의해야" 합니다");    
잘된 예
document.write('겹침 "오류를 주의해야" 합니다');
document.write("겹침 /"오류를 주의해야/" 합니다");
  • 코드를 작성할 때 중괄호{} or 소괄호()의 짝이 맞아야 합니다.
    • document.write("hi";  : (x)
document.write("hi"); : (O)
